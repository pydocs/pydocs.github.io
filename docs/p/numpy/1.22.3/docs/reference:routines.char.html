String operations

.. currentmodule:: numpy.char
    
.. module:: numpy.char

The numpy.char module provides a set of vectorized string operations for arrays of type numpy.str_ or numpy.bytes_ . All of them are based on the string methods in the Python standard library.

Comparison

Unlike the standard numpy comparison operators, the ones in the :None:None:`char` module strip trailing whitespace characters before performing the comparison.
