.. module:: numpy.testing

Test Support (:mod:`numpy.testing`)

.. currentmodule:: numpy.testing

Common test support for all numpy test scripts.

This single module should provide all the common functionality for numpy tests in a single location, so that test scripts <development-environment> can just import it and work right away. For background, see the testing-guidelines
