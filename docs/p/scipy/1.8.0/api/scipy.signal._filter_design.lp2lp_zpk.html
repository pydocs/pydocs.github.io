lp2lp_zpk(z, p, k, wo=1.0)

Return an analog low-pass filter with cutoff frequency :None:None:`wo` from an analog low-pass filter prototype with unity cutoff frequency, using zeros, poles, and gain ('zpk') representation.

Notes

This is derived from the s-plane substitution

$$s \rightarrow \frac{s}{\omega_0}$$
versionadded

Parameters

z : array_like

Zeros of the analog filter transfer function.

p : array_like

Poles of the analog filter transfer function.

k : float

System gain of the analog filter transfer function.

wo : float

Desired cutoff, as angular frequency (e.g., rad/s). Defaults to no change.

Returns

z : ndarray

Zeros of the transformed low-pass filter transfer function.

p : ndarray

Poles of the transformed low-pass filter transfer function.

k : float

System gain of the transformed low-pass filter.

Transform a lowpass filter prototype to a different frequency.
