legendre(n, monic=False)

Defined to be the solution of

$$\frac{d}{dx}\left[(1 - x^2)\frac{d}{dx}P_n(x)\right] + n(n + 1)P_n(x) = 0;$$

$P_n(x)$ is a polynomial of degree $n$ .

Notes

The polynomials $P_n$ are orthogonal over $[-1, 1]$ with weight function 1.

Parameters

n : int

Degree of the polynomial.

monic : bool, optional

If :None:None:`True`, scale the leading coefficient to be 1. Default is :None:None:`False`.

Returns

P : orthopoly1d

Legendre polynomial.

Legendre polynomial.

Examples

Generate the 3rd-order Legendre polynomial 1/2*(5x^3 + 0x^2 - 3x + 0):

>>> from scipy.special import legendre
... legendre(3)
poly1d([ 2.5,  0. , -1.5,  0. ])
