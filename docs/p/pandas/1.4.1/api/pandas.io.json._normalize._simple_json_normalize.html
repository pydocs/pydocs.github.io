_simple_json_normalize(ds: 'dict | list[dict]', sep: 'str' = '.') -> 'dict | list[dict] | Any'

Converts a nested dict into a flat dict ("record"), unlike json_normalize and nested_to_record it doesn't do anything clever. But for the most basic use cases it enhances performance. E.g. pd.json_normalize(data)

Parameters

ds : dict or list of dicts
sep : str, default '.'

Nested records will generate names separated by sep, e.g., for sep='.', { 'foo' : { 'bar' : 0 } } -> foo.bar

Returns

frame : DataFrame
d - dict or list of dicts, matching `normalised_json_object`

A optimized basic json_normalize

Examples

This example is valid syntax, but we were not able to check execution
