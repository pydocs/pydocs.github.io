filter(self, func, dropna: 'bool' = True, *args, **kwargs)

Notes

Functions that mutate the passed object can produce unexpected behavior or errors and are not supported. See gotchas.udf-mutation for more details.

Parameters

func : function

To apply to each group. Should return True or False.

dropna : Drop groups that do not pass the filter. True by default;

if False, groups that evaluate False are filled with NaNs.

Returns

filtered : Series

Return a copy of a Series excluding elements from groups that do not satisfy the boolean criterion specified by func.
