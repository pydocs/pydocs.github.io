StringDtype(storage=None)

Extension dtype for string data.

versionadded

1.0.0

warning

StringDtype is considered experimental. The implementation and parts of the API may change without warning. In particular, StringDtype.na_value may change to no longer be ``numpy.nan``.

Parameters

storage : {"python", "pyarrow"}, optional

If not given, the value of pd.options.mode.string_storage .

Attributes

None :

Methods

Examples

This example is valid syntax, but raise an exception at execution
>>> pd.StringDtype()
string[python]
This example is valid syntax, but raise an exception at execution
>>> pd.StringDtype(storage="pyarrow")
