axvspan(self, xmin, xmax, ymin=0, ymax=1, **kwargs)

The rectangle spans from xmin to xmax horizontally, and, by default, the whole y-axis vertically. The y-span can be set using ymin (default: 0) and ymax (default: 1) which are in axis units; e.g. ymin = 0.5 always refers to the middle of the y-axis regardless of the limits set by :None:None:`~.Axes.set_ylim`.
