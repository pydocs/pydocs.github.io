FigureCanvasMac(figure)

The canvas the figure renders into. Calls the draw and print fig methods, creates the renderers, etc...

Events such as button presses, mouse movements, and key presses are handled in the C code and the base class methods button_press_event, button_release_event, motion_notify_event, key_press_event, and key_release_event are called from there.
